## Java的网路移动性

**是什么？**

Java平台通过其内部机制支持在不同的主机JVM中交换 程序和数据

**为什么？**

服务商通过网路给客户机分发服务，而服务是 可执行程序和数据的集合，Java平台能的实现这种需求

**怎么样？**

Java的平台无关性和安全性是其网络移动性的基础，还通过 动态链接，class文件的紧凑性、JAR文件 来减少程序的传送时间
